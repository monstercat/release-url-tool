
# monstercat-release-url-tool

  release-url tool

## Installation

  Install with npm

    $ npm -g install monstercat/release-url-tool

## Usage

    monstercat-release-url [options]

    required:

      -f {mp3,flac,wav}
      -q {0,2} or -b {320,128}
      -r releaseId

    optional:
      -e expires   eg. 1m, 10h, 5s, 1d
